Prescription Extractor

Built an AI-powered healthcare solution that extracts structured medical information from prescriptions using OCR, NLP, OpenCV, and FastAPI while improving data accessibility and accuracy.

// Project Overview

A dynamic clinical assistance application designed to read handwritten prescriptions, isolate drug configurations, extract dosages, and translate doctor metrics into clean JSON variables. Designed to minimize pharmaceutical errors and update records swiftly.

Technical Challenges

Poorly lit uploads, slanted images, and doctor shorthand patterns corrupted traditional character classification engines.

Implemented Solutions

Integrated OpenCV Deskew filters, Contrast adjustments and custom fuzzy matching scripts referencing standard pharmacopeia catalogs.

// Core Features

AI OCR Engine

Handwriting scanner optimizing low resolution script formats.

Active Medicine Detection

Fuzzy dictionaries isolating exact chemical elements dynamically.

Dosage tracking

Parsing hourly frequency tags and cycle rates for records.

JSON Export Interface

Immediate delivery of structured records to Hospital system queues.

// Architecture Flow

1. Upload Image

Prescription photo parsed via upload modules.

2. Preprocessing & OpenCV Filters

Applying Deskew, Contrast booster, and Noise cancellation.

3. EasyOCR Text Scan & NLP Parser

Scans visual text maps, translates shapes to characters, and matches with Entity libraries.

4. FastAPI JSON Service

Routes categorized JSON attributes to the Clinic log screens.
