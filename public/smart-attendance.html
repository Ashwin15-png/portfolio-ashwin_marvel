Smart Attendance & Visitor Management System

Developed a secure attendance and visitor management platform featuring authentication, facial recognition, Dockerized deployment, CI/CD automation, and cloud-ready architecture to streamline institutional operations.

// Project Overview

An automated biometric check-in solution designed for schools, universities, or secure enterprise workspaces. Users walk in front of a camera and are instantly recognized and logged within sub-second latency using pre-computed face embedding records, saving manual tracking time and eliminating proxy attendance fraud.

Technical Challenges

Dark entry hallways and changing illumination led to poor facial recognition rates. Scaling processing streams for hundreds of simultaneous visitor arrivals created severe CPU lag.

Implemented Solutions

Introduced Histogram Equalization filters to unify brightness fields. Swapped naive comparison sweeps with pre-calculated 128D facial vectors matched via matrix calculations.

// Core Features

Facial Recognition Attendance

Accurate, instantaneous student/employee scanning using custom pre-compiled embedding indices.

Visitor Registration

Simple sign-in roster dashboard to log transient walk-ins, host mapping, and print passes.

Secure Authentication

Encrypted credential pathways protecting staff data grids and database structures.

Attendance Analytics

Dynamic CSV exports, graphic timelines, and status reports for system administrators.

// Architecture Flow

1. User Verification Request

User camera feed initiated via terminal entry points.

2. Flask Backend Router API

Directs incoming image streams to OpenCV processing pipelines.

3. OpenCV Face Recognition

Extracts pixel arrays, computes relative metrics, and queries matching codes.

4. SQLite DB Logging & Admin View

Appends time stamps, sends notification logs and updates metrics dynamically.
