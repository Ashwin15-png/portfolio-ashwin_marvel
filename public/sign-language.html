Sign Language Translation System

Designed a real-time sign language recognition system using MediaPipe and computer vision to enhance communication accessibility.

// Project Overview

An accessibility application designed to map sign gesture frames continuously using camera coordinates. Finger bones and hand joints are converted into relative vectors to match translation directories, rendering text descriptors in under 50ms latency.

Technical Challenges

Hand size scaling varied dynamically as individuals stepped closer or further away, resulting in coordinate mapping errors.

Implemented Solutions

Normalized joint distance configurations dynamically using wrist landmarks as invariant anchors, rendering matching invariant to zoom values.

// Core Features

Real-Time Detection

Continuous frame-by-frame coordinate tracking via webcam streams.

MediaPipe Joint Grid

Extracts 21 3D landmarks details for each hand skeleton instantly.

Live Translation

Dynamic text output overlays mapped to gesture configurations.

Low Latency Pipelines

Optimized ML inference running at 30+ frames per second locally.
